| 功能 >> |
| --- |

|  | title | desc |
| --- | --- | --- |
| 1 | 字符串引擎 | 可自由组合,操作字符串 |

| --- |

| title | desc |
| --- | --- |

| 张 | 想像力不必使一切与现实渲染一致,但保持一致可以使思维与行动更加一体,所以还是要尽量(目的与性能之合)设计一致; |
